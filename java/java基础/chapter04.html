1. 33

数组概述 - 静态初始化

在 Java 中，数组是一种用于存储多个相同类型数据的容器。它是编程中最基础、最常用的数据结构之一，广泛应用于数据存储、算法实现、集合操作等场景。

1. 什么是数组？

数组是一组连续内存空间，用于存储相同类型的数据。数组中的每个元素通过索引（下标）访问，索引从 0 开始。

1.1 数组的特点

  • 固定长度：数组一旦创建，长度不可变
  • 相同类型：数组中所有元素必须是同一类型
  • 连续存储：元素在内存中连续存放，访问效率高
  • 索引访问：通过索引快速访问任意元素

2. 数组的声明

数组的声明语法如下：

java
数据类型[] 数组名;

2.1 示例

java
int[] scores; // 声明一个 int 类型的数组
String[] names; // 声明一个 String 类型的数组
double[] prices; // 声明一个 double 类型的数组

> ✅ 推荐写法：数据类型[] 数组名（更清晰）

3. 数组的静态初始化

静态初始化是指在声明数组时，直接为数组元素赋值，不需要使用 new 关键字。

3.1 语法

java
数据类型[] 数组名 = {值1, 值2, 值3, ...};

3.2 示例

java
int[] scores = {90, 85, 95, 78, 88};
String[] names = {"张三", "李四", "王五"};
double[] prices = {10.5, 20.0, 15.8};

> 说明：

  • 数组长度由初始化值的个数决定
  • 不需要手动指定长度

4. 数组的访问

数组元素通过索引访问，索引从 0 开始。

4.1 示例

java
int[] scores = {90, 85, 95, 78, 88};
System.out.println(scores[0]); // 输出: 90
System.out.println(scores[2]); // 输出: 95
System.out.println(scores[4]); // 输出: 88

> ⚠️ 注意：索引越界会导致 ArrayIndexOutOfBoundsException

java
System.out.println(scores[5]); // ❌ 报错：索引越界

5. 数组的长度

数组的长度可以通过 数组名.length 获取。

5.1 示例

java
int[] scores = {90, 85, 95, 78, 88};
System.out.println("数组长度: " + scores.length); // 输出: 5

6. 静态初始化的注意事项

6.1 不能混合类型

java
int[] arr = {1, "hello"}; // ❌ 编译错误：类型不一致

6.2 不能使用变量初始化

java
int a = 10;
int[] arr = {a, 20, 30}; // ✅ 正确：变量可以使用

6.3 不能在静态初始化中使用 new

java
int[] arr = new int[]{1, 2, 3}; // ✅ 正确：等价于静态初始化

7. 总结

特性说明
静态初始化在声明时直接赋值，无需 new
语法数据类型[] 数组名 = {值1, 值2, ...};
长度由初始化值个数决定
访问通过索引 数组名[索引]
长度获取数组名.length

> 静态初始化 = 简洁、高效的数组创建方式

> 💡 建议： > - 在数组长度已知且固定时，优先使用静态初始化 > - 使用有意义的数组名（如 scores、names） > - 注意索引越界问题，避免程序崩溃 > - 在复杂逻辑中，考虑拆分为函数，提高可读性

2. 34

一维数组 - 动态初始化

在 Java 中，数组的初始化方式分为静态初始化和动态初始化。本节重点介绍动态初始化，即先声明数组，再通过 new 关键字分配内存空间，并为数组元素赋默认值。

1. 什么是动态初始化？

动态初始化是指在声明数组时，不直接赋值，而是通过 new 关键字为数组分配内存空间，并根据数据类型自动初始化元素为默认值。

1.1 语法

java
数据类型[] 数组名 = new 数据类型[数组长度];

1.2 示例

java
int[] scores = new int[5]; // 创建一个长度为 5 的 int 数组
String[] names = new String[3]; // 创建一个长度为 3 的 String 数组

> 说明：

  • 数组长度必须是整数，且不能为负数
  • 数组元素会自动初始化为默认值

2. 默认值规则

不同数据类型在动态初始化后会自动赋值为默认值：

数据类型默认值
int、byte、short、long0
float、double0.0
char'\u0000'（空字符）
booleanfalse
引用类型（如 String、Object）null

2.1 示例

java
int[] scores = new int[5];
System.out.println(scores[0]); // 输出: 0
System.out.println(scores[4]); // 输出: 0
String[] names = new String[3];
System.out.println(names[0]); // 输出: null
System.out.println(names[2]); // 输出: null

3. 动态初始化的使用场景

3.1 数组长度未知

当数组长度由用户输入或程序逻辑决定时，使用动态初始化。

java
import java.util.Scanner;
public class DynamicArrayExample {
    public static void main(String[] args) {
        Scanner scanner = new Scanner(System.in);
        System.out.print("请输入数组长度: ");
        int length = scanner.nextInt();
        int[] arr = new int[length]; // 长度由用户输入决定
        System.out.println("数组长度: " + arr.length);
    }
}

3.2 需要手动赋值

动态初始化后，可以手动为数组元素赋值。

java
int[] scores = new int[5];
scores[0] = 90;
scores[1] = 85;
scores[2] = 95;
scores[3] = 78;
scores[4] = 88;

4. 数组的访问与修改

动态初始化后，可以通过索引访问和修改数组元素。

4.1 示例

java
int[] scores = new int[5];
scores[0] = 90;
scores[1] = 85;
scores[2] = 95;
scores[3] = 78;
scores[4] = 88;
System.out.println("第 1 个成绩: " + scores[0]); // 输出: 90
System.out.println("第 3 个成绩: " + scores[2]); // 输出: 95
scores[1] = 92; // 修改第 2 个成绩
System.out.println("修改后的第 2 个成绩: " + scores[1]); // 输出: 92

5. 数组的长度

数组的长度可以通过 数组名.length 获取。

5.1 示例

java
int[] scores = new int[5];
System.out.println("数组长度: " + scores.length); // 输出: 5

> ⚠️ 注意：数组长度一旦确定，不可更改

java
scores.length = 10; // ❌ 编译错误：length 是只读属性

6. 总结

特性说明
动态初始化使用 new 分配内存，自动初始化默认值
语法数据类型[] 数组名 = new 数据类型[长度];
默认值根据数据类型自动赋值
长度由 new 指定，不可更改
访问通过索引 数组名[索引]

> 动态初始化 = 灵活、可控的数组创建方式

> 💡 建议： > - 在数组长度已知且固定时，优先使用静态初始化 > - 在数组长度由用户输入或程序逻辑决定时，使用动态初始化 > - 注意数组索引越界问题，避免程序崩溃 > - 在复杂逻辑中，考虑拆分为函数，提高可读性

一维数组 - 求最值案例

在实际编程中，经常需要对数组中的元素进行统计分析，如求最大值、最小值、平均值等。本节通过实现一个求一维数组中最大值和最小值的案例，帮助你巩固数组的遍历、条件判断和循环结构。

1. 案例目标

编写程序，输入一个一维数组，找出其中的最大值和最小值。

1.1 功能需求

  • 输入一个数组（或使用静态初始化）
  • 遍历数组，找出最大值和最小值
  • 输出最大值和最小值

2. 实现步骤

2.1 导入 Scanner 类

2.2 主程序结构

java
public class ArrayMaxMin {
    public static void main(String[] args) {
        Scanner scanner = new Scanner(System.in);
        // 输入数组
        // 求最大值和最小值
        // 输出结果
        scanner.close();
    }
}

3. 完整代码实现

java
import java.util.Scanner;
public class ArrayMaxMin {
    public static void main(String[] args) {
        Scanner scanner = new Scanner(System.in);
        // 输入数组长度
        System.out.print("请输入数组长度: ");
        int length = scanner.nextInt();
        // 动态初始化数组
        int[] arr = new int[length];
        // 输入数组元素
        System.out.println("请输入 " + length + " 个整数：");
        for (int i = 0; i < length; i++) {
            arr[i] = scanner.nextInt();
        }
        // 求最大值和最小值
        int max = arr[0];
        int min = arr[0];
        for (int i = 1; i < length; i++) {
            if (arr[i] > max) {
                max = arr[i];
            }
            if (arr[i] < min) {
                min = arr[i];
            }
        }
        // 输出结果
        System.out.println("数组: ");
        for (int i = 0; i < length; i++) {
            System.out.print(arr[i] + " ");
        }
        System.out.println();
        System.out.println("最大值: " + max);
        System.out.println("最小值: " + min);
        scanner.close();
    }
}

一维数组 - 综合案例实战

本节通过一个综合案例，将数组的声明、初始化、遍历、求最值、排序等知识点融合在一起，实现一个完整的程序：统计学生成绩并分析。

1. 案例目标

编写程序，输入多个学生的成绩，统计并输出以下信息：

  • 最高分
  • 最低分
  • 平均分
  • 及格人数
  • 不及格人数

2.2 主程序结构

    public static void main(String[] args) {
        Scanner scanner = new Scanner(System.in);
        // 输入学生人数
        // 输入成绩

        scanner.close();
    }

3. 完整代码实现

java
import java.util.Scanner;
public class StudentScoreAnalysis {
    public static void main(String[] args) {
        Scanner scanner = new Scanner(System.in);
        // 输入学生人数
        System.out.print("请输入学生人数: ");
        int n = scanner.nextInt();
        // 动态初始化数组
        int[] scores = new int[n];
        // 输入成绩
        System.out.println("请输入 " + n + " 个成绩：");
        for (int i = 0; i < n; i++) {
            scores[i] = scanner.nextInt();
        }
        // 统计分析
        int max = scores[0];
        int min = scores[0];
        int sum = 0;
        int passCount = 0;
        int failCount = 0;
        for (int i = 0; i < n; i++) {
            // 更新最大值和最小值
            if (scores[i] > max) {
                max = scores[i];
            }
            if (scores[i] < min) {
                min = scores[i];
            }
            // 累加总分
            sum += scores[i];
            // 统计及格人数
            if (scores[i] >= 60) {
                passCount++;
            } else {
                failCount++;
            }
        }
        // 计算平均分
        double average = (double) sum / n;
        // 输出结果
        System.out.println("成绩统计分析：");
        System.out.println("最高分: " + max);
        System.out.println("最低分: " + min);
        System.out.printf("平均分: %.2f%n", average);
        System.out.println("及格人数: " + passCount);
        System.out.println("不及格人数: " + failCount);
        scanner.close();
    }
}

二维数组 - 概述、定义

在 Java 中，二维数组是一种用于存储二维数据结构的容器，可以看作是“数组的数组”。它常用于表示表格、矩阵、图像等二维数据。

1. 什么是二维数组？

二维数组是一个数组，其每个元素本身也是一个数组。它通常用于表示二维数据，如表格、矩阵、棋盘等。

1.1 二维数组的特点

  • 二维结构：行和列
  • 固定大小：一旦创建，大小不可变
  • 连续存储：元素在内存中连续存放
  • 索引访问：通过 (行索引, 列索引) 访问元素

2. 二维数组的声明

二维数组的声明语法如下：

java
数据类型[][] 数组名;

2.1 示例

java
int[][] matrix; // 声明一个 int 类型的二维数组
String[][] names; // 声明一个 String 类型的二维数组
double[][] prices; // 声明一个 double 类型的二维数组

> ✅ 推荐写法：数据类型[][] 数组名（更清晰）

3. 二维数组的定义

二维数组的定义是指为数组分配内存空间，并初始化元素。

3.1 静态定义

java
int[][] matrix = {
    {1, 2, 3},
    {4, 5, 6},
    {7, 8, 9}
};

> 说明：

  • 每个内层数组是一个“行”
  • matrix[0][0] = 1, matrix[1][2] = 6

3.2 动态定义

java
int[][] matrix = new int[3][4]; // 3 行 4 列

> 说明：

  • new int[3][4] 创建一个 3×4 的二维数组
  • 所有元素默认值为 0

4. 二维数组的访问

二维数组元素通过 (行索引, 列索引) 访问。

4.1 示例

java
int[][] matrix = {
    {1, 2, 3},
    {4, 5, 6},
    {7, 8, 9}
};
System.out.println(matrix[0][0]); // 输出: 1
System.out.println(matrix[1][2]); // 输出: 6
System.out.println(matrix[2][1]); // 输出: 8

> ⚠️ 注意：索引越界会导致 ArrayIndexOutOfBoundsException

java
System.out.println(matrix[3][0]); // ❌ 报错：行索引越界

5. 二维数组的长度

二维数组的长度可以通过 数组名.length 获取。

5.1 示例

java
int[][] matrix = {
    {1, 2, 3},
    {4, 5, 6},
    {7, 8, 9}
};
System.out.println("行数: " + matrix.length); // 输出: 3
System.out.println("列数: " + matrix[0].length); // 输出: 3

> 说明：

  • matrix.length：行数
  • matrix[i].length：第 i 行的列数

6. 二维数组的初始化

6.1 静态初始化

java
int[][] matrix = {
    {1, 2, 3},
    {4, 5, 6},
    {7, 8, 9}
};

6.2 动态初始化

int[][] matrix = new int[3][4]; // 3 行 4 列，所有元素为 0

java
int[][] matrix = new int[3][];
matrix[0] = new int[2]; // 第 0 行 2 列
matrix[1] = new int[3]; // 第 1 行 3 列
matrix[2] = new int[4]; // 第 2 行 4 列

> 说明：不规则二维数组（每行长度不同）

import java.util.Scanner;
public class StudentScoreSystem {

        System.out.print("请输入班级数量: ");
        int numClasses = scanner.nextInt();
        System.out.print("请输入每个班级的学生数量: ");
        int numStudents = scanner.nextInt();
        // 动态初始化二维数组
        int[][] scores = new int[numClasses][numStudents];
        // 输入成绩
        System.out.println("请输入 " + numClasses + " 个班级，每个班级 " + numStudents + " 名学生的成绩：");
        for (int i = 0; i < numClasses; i++) {
            System.out.println("请输入第 " + (i + 1) + " 班的成绩：");
            for (int j = 0; j < numStudents; j++) {
                scores[i][j] = scanner.nextInt();
            }
        }
        // 统计分析
        int totalMax = scores[0][0];
        int totalMin = scores[0][0];
        double totalAverage = 0;
        int totalStudents = numClasses * numStudents;
        // 每个班级的平均分
        System.out.println("各班级平均分：");
        for (int i = 0; i < numClasses; i++) {
            int classSum = 0;
            for (int j = 0; j < numStudents; j++) {
                classSum += scores[i][j];
                // 更新全校最值
                if (scores[i][j] > totalMax) {
                    totalMax = scores[i][j];
                }
                if (scores[i][j] < totalMin) {
                    totalMin = scores[i][j];
                }
            }
            double classAverage = (double) classSum / numStudents;
            System.out.printf("第 %d 班平均分: %.2f%n", i + 1, classAverage);
            totalAverage += classAverage;
        }
        // 全校平均分
        totalAverage /= numClasses;
        System.out.printf("全校平均分: %.2f%n", totalAverage);
        // 每个学生的总分
        System.out.println("各学生总分：");
        for (int j = 0; j < numStudents; j++) {
            int studentTotal = 0;
            for (int i = 0; i < numClasses; i++) {
                studentTotal += scores[i][j];
            }
            System.out.printf("第 %d 学生总分: %d%n", j + 1, studentTotal);
        }
        // 输出全校最值
        System.out.println("全校最高分: " + totalMax);
        System.out.println("全校最低分: " + totalMin);
        scanner.close();
    }
}
