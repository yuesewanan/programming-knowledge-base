程序流程控制概述

程序流程控制概述

在 Java 编程中，程序流程控制是决定程序执行顺序的核心机制。它决定了代码的执行路径，使程序能够根据不同的条件做出响应，实现复杂的逻辑功能。

1. 什么是程序流程控制？

程序流程控制是指通过特定的语法结构来控制程序的执行顺序。它主要包括以下三种基本结构：

1.1 顺序结构

程序按照代码的书写顺序逐行执行，是最基本的执行方式。

java
int a = 10;
int b = 20;
int sum = a + b;
System.out.println(sum);

> 执行顺序：a = 10 → b = 20 → sum = a + b → println

1.2 选择结构（分支结构）

根据条件判断，选择执行不同的代码块。主要通过 if、if-else、if-else if 和 switch 实现。

java
int score = 85;
if (score >= 90) {
    System.out.println("优秀");
} else if (score >= 80) {
    System.out.println("良好");
} else {
    System.out.println("需努力");
}

1.3 循环结构

重复执行某段代码，直到满足退出条件。主要通过 for、while 和 do-while 实现。

java
for (int i = 0; i < 5; i++) {
    System.out.println("第 " + i + " 次循环");
}

2. 流程控制的重要性

  • 灵活性：让程序能根据输入或状态动态调整行为
  • 效率：避免重复代码，提高可维护性
  • 逻辑表达：实现复杂的业务逻辑，如用户登录、数据验证等

循环结构while条件满足时循环

跳转语句continue跳过当前循环体，进入下一次循环
跳转语句return从方法中返回

用户登录验证if-else 判断用户名密码
成绩等级评定if-else if 多级判断
数组遍历for 或 while 循环
输入验证while 循环直到输入合法
多条件菜单选择switch 语句

5. 总结

选择结构根据条件选择路径判断、分支
循环结构重复执行代码遍历、计数、等待

if分支结构

if分支结构

在 Java 中，if 分支结构是实现程序逻辑判断的基础工具，用于根据条件表达式的真假来决定是否执行某段代码。它是程序流程控制中最基本的选择结构之一。

1. if 基本语法

java
if (条件表达式) {
    // 条件为 true 时执行的代码块
}
  • 条件表达式：必须是一个布尔类型（boolean）的表达式，返回 true 或 false
  • 代码块：可以是一条语句或多个语句，用大括号 {} 包裹

示例

java
int age = 18;
if (age >= 18) {
    System.out.println("已成年，可以投票");
}

> 输出：已成年，可以投票

2. if-else 结构

当条件为 true 执行一段代码，为 false 执行另一段代码。

java
if (条件表达式) {
    // 条件为 true 时执行
} else {
    // 条件为 false 时执行
}

示例

java
int score = 75;
if (score >= 60) {
    System.out.println("考试通过");
} else {
    System.out.println("考试未通过");
}

> 输出：考试通过

3. if-else if-else 多分支结构

用于处理多个条件判断，从上到下依次判断，一旦某个条件为 true，就执行对应代码块，并跳过后续判断。

java
if (条件1) {
    // 执行代码1
} else if (条件2) {
    // 执行代码2
} else if (条件3) {
    // 执行代码3
} else {
    // 默认执行代码
}

示例

java
int score = 85;
if (score >= 90) {
    System.out.println("优秀");
} else if (score >= 80) {
    System.out.println("良好");
} else if (score >= 70) {
    System.out.println("中等");
} else {
    System.out.println("需努力");
}

> 输出：良好

4. 嵌套 if 结构

在 if 或 else 块中再写一个 if 结构，用于处理更复杂的逻辑。

java
if (条件1) {
    if (条件2) {
        // 执行代码
    }
}

示例

java
int age = 20;
boolean hasLicense = true;
if (age >= 18) {
    if (hasLicense) {
        System.out.println("可以开车");
    } else {
        System.out.println("已成年但无驾照，不能开车");
    }
} else {
    System.out.println("未成年，不能开车");
}

> 输出：可以开车

5. 注意事项

5.1 条件表达式必须是布尔类型

java
int a = 5;
if (a) { // ❌ 编译错误：int 不能隐式转换为 boolean
    System.out.println("a 不为 0");
}

✅ 正确写法：

java
if (a != 0) {
    System.out.println("a 不为 0");
}

5.2 单条语句可省略大括号

java
if (score >= 60)
    System.out.println("通过");

⚠️ 但建议始终使用大括号，提高代码可读性和安全性。

5.3 逻辑运算符的使用

  • &&：逻辑与（短路与）
  • ||：逻辑或（短路或）
  • !：逻辑非
java
if (age >= 18 && score >= 60) {
    System.out.println("可以参加考试");
}

switch分支-穿透性

switch分支-穿透性

在 Java 中，switch 是一种多分支选择结构，用于根据变量的值执行不同的代码块。与 if-else 不同的是，switch 支持穿透性（fall-through），这是其一个重要的特性，也是初学者容易出错的地方。

1. switch 基本语法

java
switch (表达式) {
    case 值1:
        // 代码块1
        break;
    case 值2:
        // 代码块2
        break;
    ...
    default:
        // 默认代码块
}
  • 表达式：必须是 byte、short、int、char、String（Java 7+）、enum 类型
  • case：匹配表达式的值
  • break：跳出当前 switch，防止穿透
  • default：当没有匹配的 case 时执行

2. switch 的穿透性（Fall-through）

2.1 什么是穿透性？

当某个 case 匹配成功后，如果没有 break，程序会继续执行下一个 case 的代码块，直到遇到 break 或 switch 结束。这种行为称为“穿透”。

2.2 示例：无 break 的穿透

java
int day = 2;
switch (day) {
    case 1:
        System.out.println("星期一");
    case 2:
        System.out.println("星期二");
    case 3:
        System.out.println("星期三");
    default:
        System.out.println("其他日子");
}

> 输出：

text
星期二
星期三
其他日子

> 分析：

  • day = 2，匹配 case 2
  • 因为没有 break，继续执行 case 3 和 default
  • 所以输出了所有后续的语句

3. 正确使用 switch：避免穿透

3.1 使用 break 防止穿透

java
int day = 2;
switch (day) {
    case 1:
        System.out.println("星期一");
        break;
    case 2:
        System.out.println("星期二");
        break;
    case 3:
        System.out.println("星期三");
        break;
    default:
        System.out.println("其他日子");
}

> 输出：

text
星期二

> 分析：

  • day = 2，匹配 case 2
  • 执行后遇到 break，跳出 switch

3.2 利用穿透性实现合并逻辑

在某些场景下，可以有意利用穿透性，将多个 case 合并为一个处理逻辑。

#### 示例：判断月份类型

java
int month = 12;
switch (month) {
    case 12:
    case 1:
    case 2:
        System.out.println("冬季");
        break;
    case 3:
    case 4:
    case 5:
        System.out.println("春季");
        break;
    case 6:
    case 7:
    case 8:
        System.out.println("夏季");
        break;
    case 9:
    case 10:
    case 11:
        System.out.println("秋季");
        break;
    default:
        System.out.println("无效月份");
}

> 输出：冬季

> 分析：

  • month = 12，匹配 case 12
  • 因为没有 break，但 case 1 和 case 2 没有代码，直接跳过
  • 所以执行 System.out.println("冬季")，然后遇到 break 跳出

4. 注意事项

4.1 break 是可选的，但通常必须使用

  • 如果忘记写 break，会导致意外的穿透行为
  • 在大多数情况下，每个 case 后都应加上 break

4.2 default 的位置

  • default 可以放在任意位置，但通常放在最后
  • 如果 default 前没有 break，它也会被穿透

4.3 switch 表达式类型限制

类型是否支持
int、byte、short、char✅
String（Java 7+）✅
enum✅
long、float、double❌
boolean❌

default默认分支，可选

> switch 分支 = 多条件判断的高效方式，但需注意穿透性

> 💡 建议： > - 除非有明确需求，否则每个 case 后都加上 break > - 使用 default 处理异常情况，提高健壮性 > - 在合并逻辑时，确保逻辑清晰，避免混淆 > - 在 Java 7+ 中，可使用 String 作为 switch 表达式，增强可读性

for循环

for循环

for 循环是 Java 中最常用的循环结构之一，用于重复执行某段代码，特别适合已知循环次数的场景。它结构清晰、语法简洁，是实现遍历数组、计数、重复任务等操作的核心工具。

1. for 循环语法

java
for (初始化表达式; 条件表达式; 更新表达式) {
    // 循环体代码
}

1.1 三个部分说明

部分作用示例
初始化表达式设置循环变量的初始值int i = 0
条件表达式判断是否继续循环，必须为 boolean 类型i < 10
更新表达式每次循环后更新循环变量i++

1.2 执行流程

1. 执行初始化表达式（只执行一次） 2. 判断条件表达式： - 若为 true，执行循环体 - 若为 false，跳出循环 3. 执行更新表达式 4. 回到第 2 步，重复执行

2. 基本示例

2.1 输出 1 到 10

java
for (int i = 1; i <= 10; i++) {
    System.out.println(i);
}

> 输出：

text
1
2
3
...
10

2.2 计算 1 到 10 的和

java
int sum = 0;
for (int i = 1; i <= 10; i++) {
    sum += i;
}
System.out.println("1 到 10 的和是: " + sum);

> 输出：55

3. for 循环的变体

3.1 从大到小循环

java
for (int i = 10; i >= 1; i--) {
    System.out.println(i);
}

> 输出：10 到 1

3.2 跳跃循环（步长为 2）

java
for (int i = 0; i < 10; i += 2) {
    System.out.println(i);
}

import java.util.Scanner;
public class LoginWithWhile {
    public static void main(String[] args) {
        Scanner scanner = new Scanner(System.in);
        String correctUser = "admin";
        String correctPass = "123456";
        int attempts = 0;

        while (attempts < 3 && !loginSuccess) {
            System.out.print("请输入用户名: ");

        }
        if (!loginSuccess) {

        }
        scanner.close();
    }
}

在 Java 编程中，死循环（Infinite Loop） 是指程序进入一个无法正常退出的循环结构，导致程序一直重复执行某段代码，无法继续后续逻辑。死循环通常由错误的条件判断或逻辑设计引起，是程序调试中常见的问题。

1. 什么是死循环？

死循环是指循环条件始终为 true，导致循环体无限执行，程序无法正常结束。

1.1 常见原因

  • 循环变量未更新
  • 条件表达式写错
  • 使用了错误的逻辑运算符

2. 死循环示例

2.1 循环变量未更新

java
int i = 1;
while (i <= 10) {
    System.out.println(i);
    // i++ 缺失，导致 i 始终为 1
}

> 输出：1 无限打印，程序卡死

2.2 条件表达式写错

java
int i = 1;
while (i > 0) {
    System.out.println(i);
    i++;
}

> 输出：1, 2, 3, ... 无限递增，直到内存溢出

2.3 使用了错误的逻辑运算符

java
int i = 1;
while (i != 10) {
    System.out.println(i);
    i++;
}

> 输出：1, 2, 3, ... 无限递增，因为 i 从 1 开始，每次加 1，永远不会等于 10？不，会等于 10，所以不会死循环。

✅ 正确写法：

java
while (i < 10) {
    System.out.println(i);
    i++;
}

3. 如何避免死循环？

3.1 确保循环变量更新

在循环体中，必须确保循环变量在每次迭代中发生变化。

java
int i = 1;
while (i <= 10) {
    System.out.println(i);
    i++; // ✅ 必须更新
}

3.2 检查循环条件

确保循环条件最终会变为 false。

java
int i = 1;
while (i <= 10) {
    System.out.println(i);
    i++;
}

3.3 使用调试工具

在 IDE 中设置断点，逐步执行代码，观察循环变量的变化。

4. 死循环的处理方式

4.1 使用 break 强制退出

java
while (true) {
    System.out.println("无限循环");
    break; // ✅ 强制退出
}

4.2 使用 Ctrl + C 终止程序

在控制台中，按 Ctrl + C 可以终止正在运行的 Java 程序。

跳转关键字 - continue 和 break

在 Java 中，continue 和 break 是两个重要的跳转关键字，用于控制循环的执行流程。它们可以改变程序的正常执行顺序，实现更灵活的逻辑控制。

1. break 关键字

1.1 作用

break 用于立即终止当前循环或 switch 结构，并跳出到循环或 switch 之后的代码。

1.2 使用场景

  • 在 for、while、do-while 循环中跳出循环
  • 在 switch 中跳出当前 case

1.3 示例

#### 示例 1：在 for 循环中使用 break

java
public class BreakExample {
    public static void main(String[] args) {
        for (int i = 1; i <= 10; i++) {
            if (i == 5) {
                break; // 当 i == 5 时跳出循环
            }
            System.out.println(i);
        }
        System.out.println("循环结束");
    }
}

> 输出：

text
1
2
3
4
循环结束

> 分析：

  • 当 i = 5 时，执行 break，跳出 for 循环
  • 不再执行后续的 i = 6 到 10
#### 示例 2：在 switch 中使用 break

java
public class SwitchBreak {
    public static void main(String[] args) {
        int choice = 2;
        switch (choice) {
            case 1:
                System.out.println("选项 1");
                break;
            case 2:
                System.out.println("选项 2");
                break;
            case 3:
                System.out.println("选项 3");
                break;

        }
    }

> 分析：

  • case 2 匹配成功，执行后遇到 break，跳出 switch
  • 不会继续执行 case 3 和 default

public class Calculator {

        Scanner scanner = new Scanner(System.in);

        while (continueCalc.equalsIgnoreCase("y")) {
            System.out.print("请输入第一个数字: ");
            double num1 = scanner.nextDouble();

            String operator = scanner.next();

            switch (operator) {

                    result = num1 + num2;
                    break;
                case "-":
                    result = num1 - num2;
                    break;
                case "*":
                    result = num1 * num2;
                    break;
                case "/":
                    if (num2 == 0) {
                        System.out.println("错误：除数不能为 0");
                        validOperation = false;
                    } else {
                        result = num1 / num2;
                    }
                    break;

                    System.out.println("错误：不支持的运算符");
                    validOperation = false;
            }
            if (validOperation) {
                System.out.printf("%.2f %s %.2f = %.2f%n", num1, operator, num2, result);
            }
            System.out.print("是否继续计算？(y/n): ");
            continueCalc = scanner.next();
        }
        System.out.println("感谢使用计算器，再见！");
        scanner.close();
    }
}

public class VerificationCode {

        Scanner scanner = new Scanner(System.in);

        }

    // 生成指定长度的随机验证码
    public static String generateCode(int length) {
        String chars = "ABCDEFGHIJKLMNOPQRSTUVWXYZabcdefghijklmnopqrstuvwxyz0123456789";
        Random random = new Random();
        StringBuilder code = new StringBuilder();
        for (int i = 0; i < length; i++) {
            int index = random.nextInt(chars.length());
            code.append(chars.charAt(index));
        }
        return code.toString();
    }
}

import java.util.Scanner;
public class PrimeFinder {
    public static void main(String[] args) {
        Scanner scanner = new Scanner(System.in);
        while (true) {
            System.out.print("请输入最大值（输入 0 退出）: ");
            int max = scanner.nextInt();
            if (max == 0) break;
            System.out.println("2 到 " + max + " 之间的素数有：");
            findPrimes(max);
        }
        scanner.close();
    }
    // 找出从 2 到 max 之间的所有素数
    public static void findPrimes(int max) {
        for (int i = 2; i <= max; i++) {
            if (isPrime(i)) {
                System.out.print(i + " ");
            }
        }
        System.out.println(); // 换行
    }
    // 判断一个数是否为素数
    public static boolean isPrime(int n) {
        if (n < 2) return false;
        if (n == 2) return true;
        if (n % 2 == 0) return false;
        // 只需检查到 sqrt(n)
        for (int i = 3; i * i <= n; i += 2) {
            if (n % i == 0) {
                return false;
            }
        }
        return true;
    }
}
