基础语法

Java
public class HelloWorld {
    public static void main(String[] args) {
        // 变量声明
        int age = 25;
        double price = 19.99;
        boolean isValid = true;
        String name = "Java";

        // 控制流
        if (age >= 18) {
            System.out.println("成年");
        }

        // 循环
        for (int i = 0; i < 5; i++) {
            System.out.println(i);
        }

        // switch 表达式 (Java 14+)
        String result = switch (day) {
            case "MON", "FRI" -> "工作日";
            case "SAT", "SUN" -> "周末";
            default -> "未知";
        };

        // 数组
        int[] numbers = {1, 2, 3, 4, 5};
        for (int n : numbers) {       // 增强 for 循环
            System.out.println(n);
        }
    }
}
📘 说明Java 是强类型语言，所有变量必须先声明类型。入口方法是 public static void main(String[] args)。

面向对象编程

Java 是彻底的面向对象语言，一切皆对象。

Java
// 封装 - 使用 private 和 getter/setter
public class Student {
    private String name;
    private int age;

    public Student(String name, int age) {  // 构造方法
        this.name = name;
        this.age = age;
    }

    public String getName() { return name; }
    public void setName(String name) { this.name = name; }
}

// 继承
public class Animal {
    public void speak() {
        System.out.println("动物叫");
    }
}

public class Dog extends Animal {
    @Override
    public void speak() {              // 方法重写
        System.out.println("汪汪!");
    }
}

// 接口
public interface Drawable {
    void draw();                    // 抽象方法
    default void print() {         // 默认方法 (Java 8+)
        System.out.println("打印");
    }
}

public class Circle implements Drawable {
    @Override
    public void draw() {
        System.out.println("画圆");
    }
}

// 抽象类
public abstract class Shape {
    public abstract double area();
}

集合框架

Java 集合框架提供了丰富的数据结构实现。

Java
import java.util.*;

public class CollectionDemo {
    public static void main(String[] args) {
        // List - 有序可重复
        List<String> list = new ArrayList<>();
        list.add("Apple");
        list.add("Banana");
        list.get(0);           // 通过索引访问
        list.remove("Apple");

        // Set - 无序不重复
        Set<Integer> set = new HashSet<>();
        set.add(1);
        set.add(2);
        set.contains(1);      // true

        // Map - 键值对
        Map<String, Integer> map = new HashMap<>();
        map.put("Alice", 90);
        map.get("Alice");     // 90
        for (Map.Entry<String, Integer> entry : map.entrySet()) {
            System.out.println(entry.getKey() + ": " + entry.getValue());
        }

        // Queue - 队列
        Queue<String> queue = new LinkedList<>();
        queue.offer("first");   // 入队
        queue.poll();             // 出队

        // Stream API (Java 8+)
        list.stream()
            .filter(s -> s.startsWith("A"))
            .map(String::toUpperCase)
            .forEach(System.out::println);
    }
}

异常处理

Java
// try-catch-finally
try {
    int result = 10 / 0;
} catch (ArithmeticException e) {
    System.out.println("除数不能为0: " + e.getMessage());
} catch (Exception e) {
    e.printStackTrace();
} finally {
    System.out.println("始终执行");  // 关闭资源
}

// try-with-resources (Java 7+) - 自动关闭资源
try (BufferedReader br = new BufferedReader(
        new FileReader("file.txt"))) {
    String line = br.readLine();
} catch (IOException e) {
    e.printStackTrace();
}

// 自定义异常
public class InsufficientBalanceException extends Exception {
    public InsufficientBalanceException(String message) {
        super(message);
    }
}

// throw 和 throws
public void withdraw(double amount)
        throws InsufficientBalanceException {
    if (amount > balance) {
        throw new InsufficientBalanceException("余额不足");
    }
    balance -= amount;
}
💡 最佳实践使用 try-with-resources 自动关闭资源；捕获具体异常而非 Exception；自定义异常需要继承 Exception（受检）或 RuntimeException（非受检）。
