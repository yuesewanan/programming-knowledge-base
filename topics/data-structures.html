📋

数组

连续内存空间，支持随机访问。适合索引频繁、插入删除少的场景

🔗

链表

非连续存储，通过指针串联。适合频繁插入删除、无需随机访问

📚

栈

后进先出 (LIFO)。用于函数调用、括号匹配、撤销操作

🚶

队列

先进先出 (FIFO)。用于 BFS、任务调度、消息队列

🗺️

哈希表

键值映射，平均 O(1) 查询。用于缓存、去重、计数

🌳

树

层次结构。用于文件系统、HTML DOM、数据库索引

排序算法

排序是最基础的算法之一。不同排序算法在不同场景下各有优劣。

算法平均时间复杂度最坏空间稳定
冒泡排序O(n²)O(n²)O(1)✓
选择排序O(n²)O(n²)O(1)✗
插入排序O(n²)O(n²)O(1)✓
归并排序O(n log n)O(n log n)O(n)✓
快速排序O(n log n)O(n²)O(log n)✗
堆排序O(n log n)O(n log n)O(1)✗
Python - 快速排序
def quick_sort(arr):
    """快速排序 - 分治思想"""
    if len(arr) <= 1:
        return arr

    pivot = arr[len(arr) // 2]  # 选择基准
    left = [x for x in arr if x < pivot]
    middle = [x for x in arr if x == pivot]
    right = [x for x in arr if x > pivot]

    return quick_sort(left) + middle + quick_sort(right)

# 示例
print(quick_sort([3, 6, 8, 10, 1, 2, 1]))
# [1, 1, 2, 3, 6, 8, 10]
Python - 归并排序
def merge_sort(arr):
    if len(arr) <= 1:
        return arr

    mid = len(arr) // 2
    left = merge_sort(arr[:mid])
    right = merge_sort(arr[mid:])

    return merge(left, right)

def merge(left, right):
    result = []
    i = j = 0

    while i < len(left) and j < len(right):
        if left[i] <= right[j]:
            result.append(left[i])
            i += 1
        else:
            result.append(right[j])
            j += 1

    return result + left[i:] + right[j:]
💡 推荐实际开发中使用语言内置排序（Python 的 sorted()、Java 的 Arrays.sort()、JS 的 arr.sort()），它们都经过高度优化。

搜索算法

Python - 二分搜索
def binary_search(arr, target):
    """二分搜索 - 要求数组已排序 O(log n)"""
    left, right = 0, len(arr) - 1

    while left <= right:
        mid = (left + right) // 2

        if arr[mid] == target:
            return mid
        elif arr[mid] < target:
            left = mid + 1
        else:
            right = mid - 1

    return -1  # 未找到

# DFS (深度优先搜索) - 递归遍历
def dfs(graph, node, visited=None):
    if visited is None:
        visited = set()
    visited.add(node)
    print(node, end=' ')

    for neighbor in graph[node]:
        if neighbor not in visited:
            dfs(graph, neighbor, visited)

# BFS (广度优先搜索)
from collections import deque

def bfs(graph, start):
    visited = {start}
    queue = deque([start])

    while queue:
        node = queue.popleft()
        print(node, end=' ')

        for neighbor in graph[node]:
            if neighbor not in visited:
                visited.add(neighbor)
                queue.append(neighbor)

树结构

树是一种分层数据结构，在计算机科学中有广泛应用。

Python - 二叉树
class TreeNode:
    def __init__(self, val=0, left=None, right=None):
        self.val = val
        self.left = left
        self.right = right

# 二叉树遍历
def inorder(root):         # 中序: 左-根-右
    if not root:
        return
    inorder(root.left)
    print(root.val, end=' ')
    inorder(root.right)

def preorder(root):        # 前序: 根-左-右
    if not root:
        return
    print(root.val, end=' ')
    preorder(root.left)
    preorder(root.right)

def postorder(root):       # 后序: 左-右-根
    if not root:
        return
    postorder(root.left)
    postorder(root.right)
    print(root.val, end=' ')

# 二叉搜索树 (BST) 搜索
def search_bst(root, target):
    if not root or root.val == target:
        return root
    if target < root.val:
        return search_bst(root.left, target)
    return search_bst(root.right, target)

动态规划

DP 通过将复杂问题分解为子问题并缓存结果来优化计算。核心思想：最优子结构 + 重叠子问题。

Python - 经典 DP
# 1. 斐波那契数列 - DP 入门
def fib(n):
    if n <= 1:
        return n
    dp = [0] * (n + 1)
    dp[1] = 1
    for i in range(2, n + 1):
        dp[i] = dp[i-1] + dp[i-2]
    return dp[n]

# 2. 0/1 背包问题
def knapsack(weights, values, capacity):
    n = len(weights)
    dp = [[0] * (capacity + 1) for _ in range(n + 1)]

    for i in range(1, n + 1):
        for w in range(capacity + 1):
            if weights[i-1] <= w:
                dp[i][w] = max(
                    values[i-1] + dp[i-1][w - weights[i-1]],
                    dp[i-1][w]
                )
            else:
                dp[i][w] = dp[i-1][w]
    return dp[n][capacity]

# 3. 最长公共子序列 (LCS)
def lcs(text1, text2):
    m, n = len(text1), len(text2)
    dp = [[0] * (n + 1) for _ in range(m + 1)]

    for i in range(1, m + 1):
        for j in range(1, n + 1):
            if text1[i-1] == text2[j-1]:
                dp[i][j] = dp[i-1][j-1] + 1
            else:
                dp[i][j] = max(dp[i-1][j], dp[i][j-1])
    return dp[m][n]
💡 DP 解题四步法
  1. 定义状态：dp[i] 表示什么
  2. 状态转移：dp[i] = f(dp[i-1], dp[i-2], ...)
  3. 初始化：确定 base case
  4. 返回值：确定最终答案的位置
