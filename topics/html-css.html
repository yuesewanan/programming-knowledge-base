HTML 基础标签

HTML（超文本标记语言）使用标签来定义网页内容的结构。以下是构建网页最常用的标签。

HTML 
<!DOCTYPE html>
<html lang="zh-CN">
<head>
  <meta charset="UTF-8">
  <title>页面标题</title>
</head>
<body>
  <h1>一级标题</h1>
  <h2>二级标题</h2>
  <p>这是一个段落文本。段落会自动换行并带有上下间距。</p>
  <a href="https://example.com">这是一个链接</a>
  <img src="image.jpg" alt="图片描述">
  <ul>
    <li>无序列表项 1</li>
    <li>无序列表项 2</li>
  </ul>
  <ol>
    <li>有序列表项 1</li>
    <li>有序列表项 2</li>
  </ol>
</body>
</html>
💡 提示 <h1> 到 <h6> 表示六级标题，每个页面建议只有一个 <h1>。<img> 的 alt 属性对无障碍访问和 SEO 非常重要。
📝

块级元素

<div>, <p>, <h1>-<h6>, <ul>, <ol>, <section> — 独占一行，宽度占满父容器

📄

行内元素

<span>, <a>, <img>, <strong>, <em> — 不会换行，宽度由内容决定

HTML 表单与语义化

表单是用户交互的核心。HTML5 引入了新的输入类型和语义标签，让网页结构更加清晰。

HTML 
<!-- 语义化标签 -->
<header>页头</header>
<nav>导航栏</nav>
<main>
  <article>
    <section>章节内容</section>
  </article>
  <aside>侧边栏</aside>
</main>
<footer>页脚</footer>

<!-- 表单示例 -->
<form>
  <label for="email">邮箱：</label>
  <input type="email" id="email" required>

  <label for="password">密码：</label>
  <input type="password" id="password" minlength="6">

  <select>
    <option value="1">选项一</option>
    <option value="2">选项二</option>
  </select>

  <button type="submit">提交</button>
</form>

CSS 选择器

CSS 选择器决定样式应用于哪些元素。掌握选择器是编写高效 CSS 的关键。

选择器示例说明
元素选择器p { }选中所有 <p> 元素
类选择器.class { }选中 class="class" 的元素
ID 选择器#id { }选中 id="id" 的唯一元素
属性选择器[type="text"]选中指定属性值的元素
伪类:hover, :first-child选中特殊状态的元素
伪元素::before, ::after选中元素的特定部分
后代选择器div p { }选中 div 内所有 p
子选择器div > p { }选中 div 的直接子元素 p
CSS 
/* 基础选择器 */
h1 { color: #333; }
.container { max-width: 1200px; }
#header { background: #f0f0f0; }

/* 组合选择器 */
div > p { /* 直接子元素 */ }
h1 + p { /* 相邻兄弟 */ }
input[type="text"] { /* 属性选择 */ }

/* 伪类 */
a:hover { color: blue; }
li:nth-child(odd) { background: #f9f9f9; }

/* 伪元素 */
.card::before {
  content: "★";
  color: gold;
}

盒模型与定位

CSS 盒模型是所有布局的基础。每个元素都被视为一个矩形盒子。

┌───────────────────────────┐ │ margin (外边距) │ │ ┌─────────────────────┐ │ │ │ border (边框) │ │ │ │ ┌───────────────┐ │ │ │ │ │ padding (内距) │ │ │ │ │ │ ┌─────────┐ │ │ │ │ │ │ │ content │ │ │ │ │ │ │ │ 内容 │ │ │ │ │ │ │ └─────────┘ │ │ │ │ │ └───────────────┘ │ │ │ └─────────────────────┘ │ └───────────────────────────┘
CSS 
/* 盒模型设置 */
.box {
  width: 200px;
  height: 100px;
  padding: 16px;    /* 内边距 */
  border: 1px solid #ccc; /* 边框 */
  margin: 20px;     /* 外边距 */

  /* 推荐：border-box 让 width/height 包含 padding 和 border */
  box-sizing: border-box;
}

/* 元素定位 */
.relative  { position: relative; top: 10px; }
.absolute { position: absolute; top: 0; right: 0; }
.fixed    { position: fixed; bottom: 20px; }
.sticky   { position: sticky; top: 0; }

Flexbox 布局

弹性盒模型是一维布局方案，擅长处理行或列方向上的元素排列与对齐。

CSS 
.container {
  display: flex;              /* 启用 Flexbox */
  flex-direction: row;       /* 主轴方向：row | column */
  justify-content: center;   /* 主轴对齐 */
  align-items: center;       /* 交叉轴对齐 */
  flex-wrap: wrap;           /* 是否换行 */
  gap: 16px;                /* 间距 */
}

.item {
  flex: 1;                   /* flex-grow: 1, 平均分配空间 */
  align-self: flex-start;   /* 单个元素对齐 */
}
📌 Flexbox 常用属性速查

容器属性： display: flex, flex-direction, flex-wrap, justify-content, align-items, align-content, gap

项目属性： flex (复合), flex-grow, flex-shrink, flex-basis, align-self, order

CSS Grid 布局

Grid 是二维布局系统，同时处理行和列，适用于复杂的页面布局。

CSS 
.grid-container {
  display: grid;
  grid-template-columns: 1fr 1fr 1fr;  /* 三列等宽 */
  grid-template-rows: auto;               /* 行高自适应 */
  gap: 20px;                              /* 行列间距 */
}

/* 响应式 Grid: 自动适应列数 */
.responsive-grid {
  display: grid;
  grid-template-columns: repeat(auto-fill, minmax(250px, 1fr));
  gap: 16px;
}

/* 显式放置元素 */
.item-special {
  grid-column: 1 / 3;  /* 跨越 2 列 */
  grid-row: 1 / 2;
}
📘 Flexbox vs Grid Flexbox 适合一维布局（行或列），Grid 适合二维布局（行和列同时控制）。通常：导航栏用 Flexbox，页面整体布局用 Grid。

响应式设计

使用媒体查询让网站在不同屏幕尺寸下都有良好表现。核心原则：移动优先。

CSS 
/* 移动优先：基础样式为手机设计 */
.container {
  padding: 16px;
}

/* 平板：≥ 768px */
@media (min-width: 768px) {
  .container {
    padding: 24px;
    max-width: 720px;
    margin: 0 auto;
  }
}

/* 桌面：≥ 1024px */
@media (min-width: 1024px) {
  .container {
    max-width: 960px;
    padding: 32px;
  }
}

/* 大屏：≥ 1440px */
@media (min-width: 1440px) {
  .container { max-width: 1200px; }
}

CSS 动画

CSS 动画包括过渡（transition）和关键帧动画（animation），可创建流畅的交互效果。

CSS 
/* 过渡: 状态改变时平滑过渡 */
.button {
  background: #4f6ef7;
  transition: background 0.3s ease,
              transform 0.2s ease;
}
.button:hover {
  background: #3a54d4;
  transform: translateY(-2px);
}

/* 关键帧动画 */
@keyframes fadeInUp {
  from {
    opacity: 0;
    transform: translateY(20px);
  }
  to {
    opacity: 1;
    transform: translateY(0);
  }
}

.animated {
  animation: fadeInUp 0.6s ease forwards;
}
属性说明
transition元素状态变化时的过渡效果
animation独立于状态变化的持续动画
transform平移、旋转、缩放、倾斜
@keyframes定义动画的关键帧序列
animation-timing-functionease, linear, ease-in-out, cubic-bezier
